
# Casical
エンジニア採用人事のためのアプリケーション「Casical」は2021年11月6日〜2021年11月7日に開催された株式会社サポーターズ主催の[技育CAMP](https://talent.supporterz.jp/geekcamp/2021/)で「エンジニアの技術力を可視化するアプリケーション」というテーマのもと作成したアプリケーションです。
